### 🚀 Quick Start for Users

```bash
# Clone the repository
git clone https://github.com/arturict/any2pdf.git
cd any2pdf

# Setup (one-time)
./setup.sh

# Convert and merge
source venv/bin/activate
python document_to_pdf.py /path/to/files --merge

# Upload merged_all_documents.pdf to ChatGPT!
```
